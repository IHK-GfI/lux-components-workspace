## Theme

Das LUX-Theme ist ein eigenständiges Projekt [lux-components-theme](https://github.com/IHK-GfI/lux-components-theme) und wird als Abhängigkeit ins LUX-Componentsprojekt eingebunden.

## Config

Die LUX-Components können über die globale Konfiguration angepasst werden.
Details befinden sich auf der [Config](config-v15)-Seite.

## Layout

Es wird empfohlen, das Layout mit den Components aus dem [Layout-Module](#layout-module) zu erstellen.
Die Anwendung sollte mit [lux-card](lux‐card-v15)s und [lux-layout-card-row](lux‐layout‐card‐row-v15)s strukturiert werden.
Für Formularelemente (z.B. [lux-input](lux‐input-v15)) gibt es die [lux-layout-form-row](lux‐layout‐form‐row-v15).
