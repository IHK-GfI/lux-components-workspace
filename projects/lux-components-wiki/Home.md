Die Demo kann auch lokal ausgeführt werden:

- LUX-Components via Git herunterladen.
- In das entsprechende Verzeichnis wechseln.
- `npm install` ausführen.
- `ng serve` ausführen.
- <http://localhost:4200/> öffnen.
- Ausprobieren!
